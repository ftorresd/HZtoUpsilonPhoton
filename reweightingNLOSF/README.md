# Reweighting LO to NLO

```
runRewightingTrees.sh
```

The will be located at: 
```ls -lha outputTrees/*```

```
runRewightingHistos.sh
```

The will be located at: 
```ls -lha outputHistos/*```

```ls -lha outputHistos/bosonsSF*.root```
